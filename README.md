# ChimestryCalculator

# Пользование
## Windows
> cd ChimestryCalculator
>python Calculator.py
## Linux
> cd ChimestryCalculator
> python3 Calculator.py
# Описание
> RU: Показывает молярную массу химического соединения.
> EN: Show the molar mass of chimetry formula.
# Пример формулы
> H2O - вода  
> H2SO4 - серная кислота
